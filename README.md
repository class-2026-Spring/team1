# 러닝코스 맞춤형 추천 서비스

## 프로젝트 소개
사용자의 운동 목적, 거리, 난이도, 선호 환경 등을 기반으로
맞춤형 러닝코스를 추천하는 서비스이다.

## 주요 기능
- 사용자 조건 입력
- 거리 기반 추천
- 난이도 기반 추천
- 날씨 기반 추천
- 시간대 기반 추천

## 기술 스택
- Python
- Pandas
- GitHub

## 추천 알고리즘
사용자 입력 조건과 코스 데이터를 비교하여 점수를 계산한 후
가장 적합한 러닝코스를 추천한다.

## 팀원 역할
- 데이터 수집
- 추천 알고리즘 설계
- PPT 제작
- GitHub 관리
