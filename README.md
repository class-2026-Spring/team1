## 실행 예시

입력:
- 난이도: easy
- 선호 환경: ocean

출력:
추천 러닝코스
이호테우 / 추천 점수: 80
삼양해변 / 추천 점수: 30

---

## 추천 알고리즘

사용자의 입력값과 코스 데이터를 비교하여
조건별 점수를 부여한다.

- 난이도 일치: +50점
- 선호 환경 일치: +30점

최종 점수를 기준으로
가장 적합한 러닝코스를 추천한다.

---

## 향후 확장 기능

- 실시간 날씨 API 연동
- GPS 기반 거리 계산

- AI 기반 개인화 추천
- 제주 지역 러닝코스 데이터 확장
