
# 러닝코스 맞춤형 추천 서비스

## 프로젝트 소개

사용자의 운동 목적, 거리, 난이도, 선호 환경 등을 기반으로  
맞춤형 러닝코스를 추천하는 서비스이다.

---

## 주요 기능

- 사용자 조건 입력
- 거리 기반 추천
- 난이도 기반 추천
- 날씨 기반 추천
- 시간대 기반 추천

---

## 기술 스택

- Python
- Pandas
- GitHub

---

## 추천 알고리즘

사용자의 입력값과 코스 데이터를 비교하여  
조건별 점수를 부여한다.

- 난이도 일치: +50점
- 선호 환경 일치: +30점

최종 점수를 기준으로  
가장 적합한 러닝코스를 추천한다.

---

## 실행 예시

입력:
- 난이도: easy
- 선호 환경: ocean

출력:

```python
추천 러닝코스
이호테우 / 추천 점수: 80
삼양해변 / 추천 점수: 30
```
